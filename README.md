# Controle de Slides via Rede usando TCP

Projeto desenvolvido para a disciplina de Redes de Computadores.

## Tecnologias
- C++
- TCP (Sockets - Winsock)
- HTML, CSS e JavaScript

## Arquitetura
Modelo Cliente-Servidor utilizando TCP na camada de aplicação.

## Execução
1. Compile e execute o servidor
2. Abra o index.html no navegador
3. Informe o IP do servidor
4. Controle os slides remotamente
